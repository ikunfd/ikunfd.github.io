项目提交Git流程

Gitee 是一款功能强大的代码托管平台，可以帮助开发者存储和管理项目代码。

安装git

  1. 前往官网链接下载Gitee, 下载64位的即可 ———-> https://git-scm.com/download/win/

  2. 下载之后安装， 一直点next最后点finish完成即可

登录注册

  1. 进入Gitee官网链接: https://gitee.com/ 注册账号

  2. 注册登录以后设置连接的邮箱，因为git命令连接时必须用到邮箱

  3. 创建空白文件夹作为本地仓库以存放项目

  4. 打开Git Bash，用命令进入本地仓库文件夹，右键菜单也会出现相应的菜单。

  5. 进入Git Bash后配置用户名和邮箱

  6. 检查是否成功

配置远程仓库

  1. 在gitee中新建仓库, 仓库名字和路径自己设置

  2. 建好远程仓库之后，在之前本地创建的空白文件夹中，创建一个新的文件夹，作为本地仓库

  3. 进入创建的本地仓库中，在空白处右击选择Git Bash Here

  4. 进入到该目录打开本地文件夹查看、显示、隐藏文件设置看到.git文件出现, .git文件是项目本地缓存区

  5. 使用git add 文件名 或者 git add .(提交所有文件)命令， 将所有文件提交到本地缓存区

  6. 将本地缓存内容提交到仓库， 使用git commit -m '提交备注信息'命令

  7. 链接仓库使用git remote add origin 仓库ssh地址命令链接仓库

  8. 然后推到远程git仓库主分支使用git push -u origin master， 就成功了

Git常用语法

  • git clone 克隆项目到本地仓库
  • git add 将工作区的修改提交到暂存区
  • git commit 将暂存区的修改提交到当前分支
  • git reset 回退到某一个版本
  • git pull 从远程更新代码
  • git push 将本地代码更新到远程分支上
  • git log 查看提交历史
